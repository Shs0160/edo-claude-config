# EDO 변경 이력

릴리즈된 변경사항을 역순(최신 상단)으로 누적.
`/release-notes` 커맨드는 이 파일의 최상단 기록 시점을 기준으로 "지난 릴리즈 이후 변경" 범위를 결정.

실제 산출물 릴리즈노트는 `edo_docs/docs/06.internal/01.planning/01.release_note.md` — 이 파일은 **Claude 내부 캐시 성격**으로 동일 내용을 간결히 누적.
