# EDO 문의 이력

고객/영업/내부에서 들어온 문의와 답변 패턴을 누적.
같은 문의가 다시 들어오면 Claude가 여기서 답변 초안을 꺼내 사용.

확정 답변 = 그대로. 초안 상태 = 제목 앞에 `[초안]` 표기.
